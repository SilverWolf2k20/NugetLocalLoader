Все заметные изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [ведении журнала изменений](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [семантического управления версиями](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-02-08

### Добавлено

- Добавлена обработка исключений ненайденного пакета.

### Изменено

- Изменен цвет вывода ошибок.
- Изменен регистр имен скачиваемых пакетов на нижний.
- Изменен вывод имени скачиваемого пакета.
- Обновлена библиотека для работы с CLI (1.6.8 -> 1.7.2).
- Обновлена библиотека для работы с цветом в консоли (4.2.0 -> 5.0.0).

### Исправлено

- Исправлена ошибка, когда в загруженных пакетах отсутствовали метаданные.
- Исправлено описание команды загрузки.
- Исправлена ошибка, когда при отсутсвии версий программа не выводила ошибку.

## [1.0.0] - 2024-01-16

### Добавлено

- Добавлено отображение цвета при выводе сообщения.
- Добавлена ссылка на репозиторий на странице приветствия.
- Добавлен вызов исключений при использовании команд, которые работают только с подкомандами.

### Изменено

- Изменена страница приветствия.
- Изменен поток вывода информации об ошибках.
- Обновлена библиотека для работы с CLI (1.6.6 -> 1.6.8).

## [1.0.0] pre-release - 2024-01-14

### Изменено

- Обновлена библиотека для работы с CLI (1.5.7 -> 1.6.6).

## [0.4.0] beta - 2024-01-08

### Добавлено

- Добавлен поиск существующих локальных пакетов. (Команда `find storage`).
- Добавлен экспорт существующих локальных пакетов в файл. (Опция `-s` для `find storage`).
- Добавлена опция игнорирования локальных пакетов из списка (Опция `-e` для `load package`).

### Исправлено

- Исправлен вылет программы при наличии в директории пакетов файла, не являющегося пакетом.
